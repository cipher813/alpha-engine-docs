# Brand Banner Snippet

Canonical one-line banner for the top of every public repo's README. Copy-paste verbatim — consistency is the point.

## Snippet

```markdown
> Part of [**Nous Ergon**](https://nousergon.ai) — Autonomous Multi-Agent Trading System. Repo and S3 names use the underlying project name `alpha-engine`.
```

## Placement

Place immediately below the H1 title, above the badge bar.

## Why this exact wording

- *"Part of"* — signals that the repo is one module of a larger system, prompting curious readers to explore the others
- *"Nous Ergon"* link → public site → entry point for the full presentation layer
- *"Autonomous Multi-Agent Trading System"* — canonical tagline, matches resume, locked across all surfaces
- *"Repo and S3 names use the underlying project name"* — disambiguates the brand-vs-project-name split so first-time readers don't get confused

Do not paraphrase or shorten. If the wording needs to change, update it here first; downstream README updates reference this file.
